# Surf Zac
![SurfZac Icon/Logo](images/SurfZacIcon.png)
## Current Version: 26.03.0
Surf Zac is a modern browser made with Chromium Embedded Framework (CEF). Release is expected in mid-march
Under the MIT License [here](https://github.com/CodedbyZach/surf-zac/blob/release/LICENSE)


**Currently, SurfZac only exists for linux distros that support AppImages. It will be rolling out to Windows (Ear March) and then Android (Late March) in the coming few months.**

Numbering System: YY/MM/Patch (same as AskZac)
