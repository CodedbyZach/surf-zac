# Surf Zac

Surf Zac is a modern browser made with Chromium Embedded Framework (CEF). Full release is expected in mid-march
Under the MIT License [here](https://github.com/CodedbyZach/surf-zac/blob/release/LICENSE)


**Currently, SurfZac only exists for linux. It will be rolling out to Windows (Early-ish March) and then Android (Late March).**
